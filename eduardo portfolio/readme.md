# Portfólio de eduardo augusto

Este é meu portfólio pessoal, desenvolvido com HTML, CSS e JavaScript. Ele inclui seções sobre minhas habilidades, projetos e interesses.

- Navegação entre páginas (Início, Sobre Mim, Projetos).

- Layout responsivo usando Flexbox.

## Como Usar
1. Clone o repositório: git clone 
2. Abra index.html em um navegador.

## Tecnologias
- HTML5
- CSS3 (com Flexbox)
- JavaScript
